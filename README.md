# Como Rodar o Projeto
Este projeto foi criado para demonstrar um fluxo de autenticação OAuth 2.0 em Flutter, com renovação de token e listagem/criação de pedidos. Seguem as instruções de uso:

## Clone ou baixe o repositório
Faça o download do código ou clone o repositório para a sua máquina local.

## Instale as dependências
Abra o terminal na pasta do projeto e rode:

```bash
flutter pub get
```
## Execute o aplicativo
Conecte um dispositivo ou emulador e, ainda no terminal, rode:

```bash
flutter run
```
O app será iniciado exibindo a tela de splash, que verificará se existe um token salvo. Caso não exista ou seja inválido, você será direcionado para a tela de login.
